# OmniFood
This is a dummy Food Ordering System named *OmniFood*.<br>
### Technologies used
#### For Client Side:
1. HTML<br>
2. CSS<br>
3. JQuery<br>
4. Vanilla JS<br>
5. Angular<br>
 
#### For Server Side:
1. NodeJS<br>
2. ExpressJS<br>
3. MongoDB<br>

### To run on localhost:
1. Clone repository<br>
2. Run **npm install**<br>
3. Execute **node app.js**<br><br>
***You're good to go***
